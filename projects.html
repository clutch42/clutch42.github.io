MEAN Stack Project from CS-465

The CS-465 project at Southern New Hampshire University focuses on developing a full-stack web application using the MEAN stack, which includes MongoDB, Express.js, Angular, and Node.js. MongoDB serves as the NoSQL database, where data is stored in a flexible, JSON-like format, and commonly utilizes the Mongoose library for object modeling and schema validation. Express.js is used as the back-end framework, allowing for the creation of a RESTful API that handles CRUD operations and integrates middleware, using libraries like body-parser, cors, and express-validator. Angular is employed on the front end, providing a structured framework for building dynamic user interfaces with features like two-way data binding, dependency injection, and routing, often using libraries like @angular/forms, @angular/router, and rxjs. Node.js serves as the server-side runtime, executing JavaScript to manage back-end logic and interactions with the database and front-end. In addition, the project often includes user authentication using JSON Web Tokens (JWT) and password hashing with libraries such as jsonwebtoken and bcryptjs, while responsive UI design is typically achieved with Angular Material or Bootstrap. Overall, the project brings together various technologies to deliver a robust, dynamic web application.
