Project Narrative: Enhancing Efficiency

Artifact Description: This project from CS-300 Data Structures and Algorithms, completed in summer 2023, involves creating a binary search tree in C++. It parses a .csv file to manage bids, performs in-order traversal, and supports search and deletion operations, illustrating key data structure concepts.

Justification for Inclusion: The project is included in my ePortfolio to showcase my understanding of data structures and algorithms. While initially functional, the original BST lacked efficiency with large datasets. This project highlights the importance of balanced trees for optimal performance.

Improvements and Enhancements: I upgraded the BST to an AVL tree, implementing rotation operations and adjustments to maintain balance, which improved performance to O(log n). I tested its efficiency, updated the menu for dynamic input, and enhanced functionality with a flexible in-order traversal using function objects.
