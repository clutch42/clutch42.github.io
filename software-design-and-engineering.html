Project Narrative: Enhancing Code Readability

Artifact Description: This project from CS 330: Computational Graphics and Visualization, completed in Fall 2023, demonstrates the use of OpenGL and C++ to create a dynamic 3D scene with advanced textures, lighting, and shading techniques.

Justification for Inclusion: Despite achieving its initial goals, the project's original code structure was disorganized, with all vertex and index generators located in a single .cpp file. This setup made the code hard to manage and extend.

Improvements and Enhancements: Significant improvements were made by refactoring the code to move vertex and index generators into separate header files. Each shape was encapsulated into its own class, enhancing readability and security. Constructors for varying shape sizes were added, providing more flexibility for future expansions.

Course Outcomes

The enhancements made to this project effectively met the following course outcomes:

  • Collaborative Coding Environment: The refactoring and modularization efforts contributed to a more collaborative coding environment.
  • Application of Computer Science Practices: Header files and structured practices improved code maintainability and scalability.
  • Enhanced Code Security: A class-based structure and reduced global variables improved the security and robustness of the codebase.
  • Improved Code Communication: Well-documented code and clear class definitions enhanced project communication.

Reflection on the Process

The project highlighted the importance of modular design and class encapsulation in managing complex systems. These improvements made the codebase easier to understand and extend. Future projects will benefit from more thorough initial planning to avoid structural issues. The choice between retaining the scaling method or adopting new instances proved challenging, but the current setup allows flexibility for future developments. Additionally, refactoring has removed magic numbers and replaced global variables with named constants, resulting in a cleaner codebase.
